# sudoku
•	利用程序随机构造出N个已解答的数独棋盘 。
•	输入
•	数独棋盘题目个数N（0<N<=1000000）
•	输出
•	随机生成N个不重复的已解答完毕的数独棋盘，并输出到sudoku.txt中，输出格式见下输出示例。
